Real-world Maieutics sessions demonstrating the skill across different domains.

| Example | Domain | Key Logical Gap | Outcome |

| [Backend Rewrite](backend-rewrite.md) | Engineering | Solution proposed before problem understood | User discovers language choice may be irrelevant |

## Contributing Examples

Have a Maieutics session that produced clarity? Submit it as a PR:

1. Copy this template and fill in your conversation
2. Highlight the phase transitions and exit conditions
3. Include a "Post-Session Notes" section explaining what happened and what didn't
4. Ensure the agent never provides answers, solutions, or factual filler

We especially welcome examples from:
- Strategic business decisions
- Ethical dilemmas
- Technical architecture debates
- Personal relationships and communication
- Scientific hypothesis formation
